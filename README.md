# Hi there, I'm Aanandit! 👋

Welcome to the source code for my personal portfolio website! I built this from scratch to showcase my journey as a Computer Science student with a huge passion for **Cybersecurity**. 

Currently, I'm studying at VIT Vellore (Batch of 2028), and I love getting hands-on with everything from Linux systems and complex databases to building modern web interfaces. 

## 🌟 What's this project all about?

This isn't just a basic static site. I wanted it to feel alive and welcoming! So, I designed it with a sleek, dark mode aesthetic featuring "glassmorphism" (those cool, frosted-glass panels you see), subtle glowing gradients, and some fun micro-animations like the glitching text on the homepage. 

Behind the scenes, I kept things clean and straightforward:
- **The Foundation:** Classic HTML5, CSS3, and Vanilla JavaScript.
- **A Touch of React:** I embedded React 18 directly into the site (without needing a complicated build process) to handle neat interactive components, like the Connect button!
- **Media & Modals:** I built custom, interactive pop-ups to display my certifications and embedded my actual project walkthrough videos directly from Google Drive.

## 🛠️ Want to poke around the code?

Since I built this keeping the tech stack lightweight and accessible, you don't even need to install any heavy packages or deal with `npm`. 

1. Just clone the repo to your machine:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. Open the folder and double-click `index.html` to open it in your browser. That's literally it!
